... τώρα που βρήκαμε παπά, να θάψουμε πέντε-έξι?

 

Νομίζω πως τώρα μπορείς να φτιάξεις σε Delphi ένα πρόγραμμα που θα κάνει αυτόματα registration σε καμιά δεκαριά εταιρίες free hosting, θα ανοίγει το εκτελέσιμο του Dreamweaver, θα το μετατρέπει σε Javascript, και θα ανεβάζει με ftp μερικά site που θα δημιουργούνται μόνα τους! α - και μην ξεχάσεις να βάλεις ως bonus την αυτόματη δημιουργία ενός Flash που θα ζωγραφίζει τρίγωνα, τετράγωνα, πεντάγωνα, οκτάγωνα, τετράεδρα, πολύγωνα, πολύεδρα, πλατωνικά στερεά, τη μοριακή δομή του LSD, και τη Mona Liza να χαμογελάει, θα τετραγωνίζει τον κύκλο και θα υπολογίζει την απόσταση της γης από το φεγγάρι - με κανόνα και διαβήτη.

Αν (παρ'ελπίδα) τα παραπάνω αποτύχουν μπορείς :

α) να αλλάξεις γλώσσα προγραμματισμού (μπορείς να δοκιμάσεις σε Visual Fortran, Java 5 ή C++--**//.NET)
β) να γυρίσεις στα παλιά σου λημέρια (hardware) και να φτιάξεις επιτέλους με fuzzy logic και τροφοδοσία από καμένες πευκοβελόνες ή αλατισμένο χιόνι, τον αυτόματο ελεγκτή αερόθερμου που πάντα επιθυμουσες!

Ο χρόνος σας μετράει από.......τώρα! Good luck, Jim
