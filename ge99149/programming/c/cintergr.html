Ολοκλήρωση σε C

Αν αρωτηθήκατε ποτέ πως μπορούν τα κομπιουτεράκια και υπολογίζουν ολοκληρώματα με τόσο μεγάλη ακρίβεια και τόσο σύντομα, ενώ εμάς μας παίρνει ώρες; Οι υπολογιστές φυσικά δεν γνωρίζουν αναλυτικές μεθόδους. Δεν βρίσκουν δηλαδή την αρχική συνάρτηση του ολοκληρώματος κ.τ.λ. Αντίθετα χρησιμοποιούν αριθμητικές μεθόδους πολύ κοντά στον ορισμό του ολοκληρώματος.

Η παρακάτω μέθοδος ονομάζεται μέθοδος του τραπεζίου και ουσιαστικά χωρίζει την συνάρτηση σε πολύ μικρά κοματάκια dx και θεωρεί το εμβαδό τους προσεγγιστικά ίσο με αυτό ενός τραπεζίου. Η μέθοδος αυτή δίνει απίστευτα καλά αποτελέσματα και μάλιστα πολλές φορές καλύτερα από αυτά ενός συνηθισμένου computer. Η συνάρτηση που θέλετε να ολοκληρώσετε δηλώνεται ως float f(float x) και μπορείτε να την τροποποιήσετε σε οποιαδήποτε συνάρτηση εσείς θέλετε.
Ο αλγόριθμος είναι από το ηλεκτρονικό βιβλίο:Numerical Recipes in C: the Art of Scientific computing (κεφάλαιο 4.2),ένα καταπληκτικό ηλεκτρονικό βοήθημα για αριθμητική ανάλυση.

Το πρόγραμμα ολοκλήρωσης σε C

#include <math.h>

float trapzd(float (*func)(float), float a, float b, int n)
{
float x,tnm,sum,del;
static float s;
int j,it;

if (n == 1) {
return (s=0.5*(b-a)*(func(a)+func(b)));

} else {
for (it=1,j=1;j<n-1;j++) it <<= 1;
tnm=it;
del=(b-a)/tnm;
x=a+0.5*del;
for (sum=0.0,j=1;j<=it;j++,x+=del) sum += func(x);
s=0.5*(s+(b-a)*sum/tnm);
return s;
}
}


float f(float x)
{
return(53*pow(x,3)-10*exp(x+2));
}

main(void)
{
int j,m;
float a,b,s;

printf("Dwse to panw kai katw orio oloklirwsis : ");
scanf("%f%f",&a,&b);
printf("Dwse ton ari8mo twn orwn oloklirwsis (mexri 14) : ");
scanf("%d",&m);

for(j=1;j<=m+1;j++) s=trapzd(f,a,b,j);

printf("\n To oloklirwma apo %3.2f ews %3.2f tis F(x) einai : %f",a,b,s);
}
