Pascal - C

Γενικό Σχόλιο: Η C είναι μία γλώσσα με πολύ περισσότερες δυνατότητες απ'την Pascal. Αυτό σημαίνει ότι απαιτείται μεγαλύτερη προσοχή.

Τη C και άλλες χρήσιμες πληροφορίες μπορείτε να βρείτε στο http://www.cc.ece.ntua.gr/~balex/

Παρατήρηση: Με τη Pascal έγινε μια προσπάθεια εισαγωγικής προσέγγισης στις γλώσσες προγραμματισμού. Δεν απαιτείται η γνώση Pascal για τον προγραμματισμό σε C.

Αντιστοιχίες Pascal - C

Το πρώτο πρόγραμμα (που χαιρετάει):

Program Pas(input,output);

  
 

int main(void)

begin;

{

write('Hello World!');

printf("Hello World!");

end.

}

Ένα λίγο ποιό σύνθετο πρόγραμμα (Χαιρετάει με το όνομά σου):

Program Pas(input,output);

  

var name : string;

char name[20];
 

int main(void)

begin;

{

write('Hello, what is your name? ');

printf("Hello, what is your name? ");

readln(name);

scanf("%s",name);

write('Hello ',name,' !');

printf("Hello %s !",name);

end

}

Ένα πρόγραμμα που... μετράει :

Program Pas(input,output);

  

var number,count : integer;

int number,count;
 

int main(void)

begin;

{

write('What number do you want to count to? ');

printf("What number do you want to count to?  ");

readln(number);

scanf("%d",&number);

for count:=1 to number do

for (count=1; count<=number; count++)

writeln(count,' ');

printf("%d \n",count);

end.

}

 

Συμπέρασμα: Υπάρχουν πάρα πολλές ομοιότητες ανάμεσα στη C και την Pascal. Η C είναι λίγο πιό λιτή αλλά και πιό ιδιόρυθμη.
π.χ. Αν παραληφθεί το '&' πρίν το number στην εντολή scanf...

 

Ασκήσεις

Με βάση τα παραπάνω μπορούν να υλοποιηθούν τα εξής :

  • Ένα πρόγραμμα που τυπώνει τις ημέρες της εβδομάδας (με απλή χρήση της printf).
  • Ένα πρόγραμμα που ρωτάει και υποστηρίζει την ομάδα σου.
  • Ένα πρόγραμμα που διαφωνεί με ότι κι αν γράφεις, π.χ.:
    >> 8elw_na_fygw
    >> Oxi, den 8elw_na_fygw
  • Ένα πρόγραμμα που μετρά ανάποδα από κάποιο αριθμό.
  • Ένα πρόγραμμα που να γράφει ένα ν*ν μοναδιαίος πίνακα: (Χρησιμοποιήστε την εντολή if (x==y) printf("1"); else printf("0"); )
  • Ένα πρόγραμμα που λέει χίλια ευχαριστώ!

Ένα πρόγραμμα που μετρά ανάποδα από κάποιο αριθμό

int number,count;

{
printf("What number do you want to down count from? ");
scanf("%d",&number);
for (count=0; count<=number; count++)
printf("%d \n",number-count);
}

int number,count;
int main(void)

for (count=number; count>=0 ; count--)
