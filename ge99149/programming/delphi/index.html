Delphi

Μιά άνετη γλώσσα προγραμματισμού

Η Delphi (την ταυτίζω με τον C++ Builder για όσους προτιμούν C αντί για Pascal) είναι κατά την γνώμη μου η καλύτερη επιλογή για όσους θέλουν να κάνουν με ευκολία προγράμματα για Windows και θέλουν να έχουν μεγαλύτερο έλεγχο από τη Visual Basic! Είναι δυνατή ευέλικτη και αρκετά εύκολη χάρη στις έτοιμες βιβλιοθήκες της Borland.
Λίγα για τη στιγμή... μερικά sources και εκτελέσιμα.
