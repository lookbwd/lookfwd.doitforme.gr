Προγραμματίζοντας σε Pascal

Στο ΤΕΜΦΕ διδάσκεται στο πρώτο εξάμηνο η γλώσσα Pascal, παρόλο που στο επόμενο εξάμηνο διδάσκεται C η οποία ως γνωστόν είναι πολύ πίο δυνατή και διαδεδομένη!

Γιατί Pascal?

  1. Η Pascal είναι μία γλώσσα εκπαιδευτικού χαρακτήρα, πιό ασφαλής απ'τη C.
  2. Αν γνωρίζεις Pascal μπορείς εύκολα να μάθεις οποιαδήποτε άλλη γλώσσα πολύ εύκολα.
  3. Ακόμα και πολύ καλοί προγραμματιστές χρησιμοποιούν (σε διαγωνισμούς) Pascal αντί για C επειδή τους επιτρέπει να γράψουν τα προγράμματα πιό γρήγορα (χωρις να έχουν την ενόχληση από τις ιδιοτροπίες της C)
  4. Η Pascal για Windows (Delphi) είναι πολύ ευκολότερη από την C , αποδοτικότερη από Visual Basic και κατά τη γνώμη μου, η καλύτερη επιλογή για εύκολο προγραμματισμό σε windows.
