Εκφώνηση: Κάντε ένα πρόγραμμα το οποίο προσθέτει δύο αριθμούς

Η εντολή := δίνει στο πρώτο μέλος την τιμή του δεύτερου. Καλό επίσης είναι ένα πρόγραμμα να είναι φιλικό (ευγενικό) με τον χρήστη.

Program prosthesi (input,output);
Var Ar1 , Ar2 ,Sum : integer;
begin;

Writeln ('8a pros8esw 2 ari8mous.'); writeln;

Write ('Dwse mou ton prwto ari8mo : ');
Read(Ar1); Readln;

Write ('Dwse mou ton deutero ari8mo : ');
Read(Ar2); Readln;
Sum := Ar1 + Ar2;

Writeln( Ar1 ,' + ', Ar2,' = ', Sum ); writeln;
Writeln ('Fxaristw ! ');
End.
